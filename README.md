формат хода: (нач.коорд. - кон.коорд mod)
mod - особенность хода (шах, мат, превращение пешки)
обозначения: 
"+" шах, "#" мат, "=" пат, q,r,k,b - превращение пешки в ферзя, ладью, коня и слона соответсвенно

пример хода: "e2-e4" , "f3-f7#" , "a7-a8q"
